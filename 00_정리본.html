글자 관련 태그


html 태그 : html의 내용을 작성하는 태그
head 태그 : html을 정의하는 내용 작성
body 태그 : 화면(브라우저)에 보여질 내용을 작성하는 태그
hr(horizon) : 수평선
br(line BReak) : 엔터
  : 띄어쓰기(스페이스)
h1~h6 : 제목(Highlight)을 나타내는 태그 1이 제일 큼, 6이 제일 작음
pre(PREformated) 태그 (사전 형식 지정) : 작성한 형식 그대로 출력하는 문단
p(Paragraph : 문단) 태그 : 글을 작성하는 용도의 태그--
b(bold) 태그 : 단순히 글자를 굵게 표시하는 태그
i(italic) 태그 : 단순히 글자를 기울여 표시하는 태그
strong 태그 : 글자(단어)를 강조하는 용도의 태그
em 태그 : 글자(문장)을 강조하는 용도의 태그
mark 태그 : 형광펜 효과를 추가하는 태그
del 태그 : 글자 가운데 선을 긋는 태그(삭제선, 이제 없음을 표시)
small 태그 : 글자를 작게 표시하는 태그
sup 태그 : 윗첨자 x3
sub 태그 : 아랫첨자 log25
abbr(ABBReviation : 약어) 태그 : 마우스 오버 시 툴팁 표시

목록 관련 태그

li(List item) 태그 : 목록의 한 줄
ul(Unordered List) 태그 : 순서가 없는 목록
ol(Ordered List) : 순서가 있는 목록

표 관련 태그

tr(Table Row(행)) 태그 : 테이블의 한 행을 나타내는 태그
td(Table Data) 태그 : 테이블 데이터를 작성할 수 있는 태그(각 행의 열 역할을 함)
th(Table Header) 태그 테이블의 제목 또는 내용을 강조할 때 td 대신 사용하는 태그
caption 태그 : 테이블에 대한 설명을 작성하는 태그
thead 태그 : 테이블 상단 영역(컬럼 제목 영역)
tbody 태그 : 테이블 중단 영역(컬럼 내용 영역)
tfoot 태그 : 테이블 하단 영역(합계, 총합 영역)
rowspan 태그 :행병합(가로로 나아감, 좌에서 우로)
colspan : 열병합(세로로 나아감, 위에서 아래로)

이미지 관련 태그

img 태그 : 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

영역 관련 태그

iframe 태그 : 웹 문서(html) 내부에 또 다른 웹 문서(html)를 추가하는 태그

하이퍼 링크 관련 태그

a(anchor : 연결) 태그 : 현재 HTML 문서와 다른 HTML 문서를 연결 시켜 클릭 시 이동할 수 있게 만드는 태그

입력 관련 태그

input 태그 : 웹 문서에서 사용자(클라이언트)가 입력할 수 있는 양식을 제공하는 태그 type="text" : 한 줄 텍스트 입력 상자 (기본값) type="password" : 비밀번호 입력 상자(모자이크 처리) type="number" : 숫자만 입력 가능한 입력 상자 type="range" : 슬라이드바를 이용해서 숫자를 지정 type="radio" : 여러 값 중 하나 선택 type="checkbox" : 여러 값 중 원하는 값 모두 선택
